# 📋 제품 출시 & 프로모션 일정 취합 시스템

## 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 서버 실행
```bash
python app.py
```

### 3. 브라우저 접속
- **제출 페이지**: http://localhost:5000
- **관리자 페이지**: http://localhost:5000/admin
- **엑셀 다운로드**: http://localhost:5000/export

---

## 기능 설명

### 제출 페이지 (각 팀 담당자용)
1. 담당팀 & 담당자 이름 입력
2. 일정, 구분, 제품명, 비고 입력 후 [항목 추가] 클릭
3. 여러 항목을 한 번에 추가 가능
4. [제출하기] 버튼으로 한꺼번에 전송

### 관리자 페이지
- 전체 제출 현황 통계 확인
- 팀별 / 구분별 / 제품명 검색 필터
- 개별 항목 삭제 가능
- 엑셀 다운로드

### 엑셀 다운로드
- `취합` 시트: 전체 데이터 (일정 순 정렬)
- 팀별 시트: 각 팀 데이터 별도 시트
- 원본 파일과 동일한 양식 (헤더, 스타일 포함)
